### âš™ï¸ Setup

> **Prerequisites**: To launch Jumper documentation, you need to have [Node.js](https://nodejs.org/) installed on your machine.

#### TL;DR (Quick Setup)

```bash
git clone https://github.com/Jumper-Carrot/Jumper-doc.git
cd Jumper-doc
npm install
npm run docs:dev
```

-----

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/Jumper-Carrot/Jumper-doc.git
    cd Jumper-doc
    ```
2.  **Install dependencies:**

    ```bash
    npm install
    ```
3.  **Run Jumper doc in development mode:**

    ```bash
    npm run docs:dev
    ```
